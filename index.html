Click on a dot to learn more about that faith-based property

The following map shows faith-based properties in Brooklyn. Each dot is one of the 2,743 faith-based properties in the borough. By clicking on the dot, users can learn more about the address, the property owner, and the amount of residential development rights associated with the tax lot. Due to file size constraints, the entire property owner names cannot be displayed. Users can either locate properties manually by scrolling through the map or by clicking on the icon in the top right corner of the map and using the search bar.

Users can learn more about the purpose of the project as well as the methodology used to calculate residential development rights in the 'About' section.

Additional data associated with each property is also available for download in both .csv and .shp formats.
